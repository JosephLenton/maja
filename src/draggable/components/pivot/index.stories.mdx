The pivot is used to offset contents in a standardised way.

 * Give it an offset
 * Put contents inside
 * Contents shows up offset! 🎉

It is used primarily with the editor.

## An example

Here is an example card offset inside of an editor.

<MajaContext>
  <ExamplePartialEditor>
    <Pivot offset={{ x: 120, y: 320 }}>
      <EditorCard>
        <EditorCardButton>😻</EditorCardButton>
        <EditorCardButton>🐶</EditorCardButton>
        <EditorCardButton>🐼</EditorCardButton>
      </EditorCard>
    </Pivot>
  </ExamplePartialEditor>
</MajaContext>

The Pivot only provides an offset. It is from the bottom left of the screen. That is all.
